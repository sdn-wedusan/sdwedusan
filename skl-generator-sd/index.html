Kelola Impor & Ekspor Massal Excel

Untuk menghemat waktu pengisian nilai menggunakan banyak laptop guru kelas, gunakan alur pengisian massal di bawah ini:

1. Unduh Template Excel

Unduh berkas Microsoft Excel (.xlsx) resmi dengan struktur kolom siap isi.

2. Unggah Berkas Excel

Unggah kembali file Excel (.xlsx) yang telah diisi lengkap dengan nilai oleh guru kelas.

3. Unduh Excel Data Siswa

Unduh seluruh data siswa aktif beserta nilai mereka dalam format Microsoft Excel (.xlsx).

Cadangkan & Pulihkan Database (JSON)

Ekspor seluruh pengaturan sekolah, logo, rumus nilai, dan daftar siswa ber-nilai lengkap menjadi satu file JSON. Amankan file ini sebagai cadangan kelulusan tahunan sekolah Anda.

Petunjuk Pengisian Impor Excel/CSV

  • Kolom spreadsheet wajib diawali oleh struktur biodata dasar: **Nama Lengkap, Agama, Tempat Lahir, Tanggal Lahir, Orang Tua Wali, NIS, NISN, Nomor Ijazah**.
  • Kolom nilai harus mengikuti pola penamaan mata pelajaran resmi, contoh: **[Mata Pelajaran] - Sem 9, [Mata Pelajaran] - US, [Mata Pelajaran] - Praktik**.
  • Unduh template Excel/CSV yang kami sediakan untuk memastikan akurasi data pemetaan kolom saat diunggah.

Edit Biodata & Nilai Siswa
